# Garbage Drawer

This folder contains old unfinished, deprecated, or ignored routed surfaces.

Do not read these files for general app orientation. Do not treat them as product direction. Open or edit them only when the user explicitly asks for a screen in this folder.

The route ids stay alive so old tools can still be reached from `Prototyping`, but the active product surfaces live in `tabs/primary` and `tabs/secondary`.

File names should match the labels shown in Prototyping:

- `CurrencyStrengthFromCandlesTab.tsx`
- `WatchlistEngineTab.tsx`
- `MacroStateTab.tsx`
- `SixQuestionsDraftTab.tsx`
- `WipMapArchiveTab.tsx`
- `StrengthMeterTab.tsx`
- `DeprecatedOverviewTab.tsx`
- `ArchivedEventReactionStudyTab.tsx`
- `ArchivedEventQualityStudyTab.tsx`
