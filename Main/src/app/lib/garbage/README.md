# Garbage Logic

This folder contains logic used by old unfinished, deprecated, or ignored routed surfaces under `tabs/garbage`.

Do not read these files for general app orientation. Do not treat them as product direction. Open or edit them only when the user explicitly asks for a garbage-drawer screen or its supporting logic.

Active Event Replay helpers intentionally remain in `app/lib`, especially `eventReaction.ts` and `eventQuality.ts`.
